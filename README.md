## Space Shooter Game

>Require: **Python 3.+**
 
---

### Dependencies: 

    $ pip install -r requirements.txt


---

### Game:

Run

    $ python game.py

---

Options:

    Steering : W A S Da
    Aiming: Mouse
    Shooting: Left Mouse Button

    Obcjetive: Survive as long as you can, get top score
